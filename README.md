# getion-d'ouvrages

``` bash

*******************************Installation*******************************************


# 1) Installer les dépendances du Backend.
    $ cd BackEnd
    $ npm install / npm i

# 2) Installer les dépendances du Frontend.
    $ cd FrontEnd
    $ npm install / npm i


******************************Lancement en mode Prod**********************************

# BACKEND
# 1) Si vous voulez travailler avec votre propre base de donnée (mpngoDB) Aller dans le fichiers ".env" et mettez les bonnes configurations dans lesquelles vous voulez travailler, sinon jai mis toutes les configuration d'une base de donnée (mongoDb) que jai créer en ligne 
# 2) Lancez la commande  : 
    $ npm start

# FRONTEND
# 1) pour modifier le proxy vous pouvez aller sur le fichier proxy.conf.json
# 2) Lancez la commande  : 
    $ npm start



******************************Version***********************************
#  en cas de problems de version mettez a jour la version
    $ node v13.3.0
    $ Angular 10

```
